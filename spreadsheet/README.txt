Die Demo der Webkomponente ist ohne eine bestehende Internetverbindung nicht möglich.
Dies liegt an den ES6 Modulen der Bibliothek RxJS, die zur Ausführung von einem Content Delivery Network geladen werden müssen.
Das laden eines ES6 Moduls über das 'file://' Protokoll ist zu dem Zeitpunkt der Abgabe der Webkomponente nicht möglich.

Der Prototyp zur Tabellenkalkulation ist für den Chrome Browser entwickelt.
Die Demo der Webkomponente ist deshalb mit dem Chrome Browser Version 80.0.3987.122 (und höher) auszuführen.

Um die Demo zu starten, führen Sie die demo.html Datei mit ihrem Chrome Browser aus.

Für die Auswahl mehrerer Zellen wählen Sie eine Startzelle durch einen Klick mit der linken Maustaste aus. 
Danach wird die Endzelle mit gedrückter Shift-Taste und erneuter Betätigung der linken Maustaste ausgewählt.
In der oberen linken Zelle der Kontrollleiste wird der aktuell selektierte Zellenbereich angezeigt.
